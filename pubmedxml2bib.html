Try it out

This is a demonstration of pubmedxml2bib embedded in a website. To try it out, go to PubMed, perform an arbitrary search, open any of the search results. Then click on Format: Abstract in the top left corner of the displayed result and select XML. Copy the XML output, paste it into the left text area above, and click on Convert to BibTeX.

Alternatively, especially if you would like to export several search results, select all results to be exported, click on Send to in the top right corner, select File and choose XML under Format. Once you saved the file, you can open it with any text editor and paste the contents into the left text area above. If you frequently export search results to files or if you store references locally in PubMed XML format, you may prefer to use the command line version of pubmedxml2bib instead of this website. Kotlin source code as well as JVM binaries and scripts can be downloaded from the pubmedxml2bib project website.

Purpose of this Software

PubMed, the search engine for scientific articles from biomedical and other fields, offers its search results in a detailed and structured XML format. PubMed XML is an ideal format for storing and automatically processing databases of citations. pubmedxml2bib can be used to convert this XML format into BibTeX, either as a command line script (requires a Java Virtual Machine) or embedded into a website, such as here. The latter functionality allows, for instance, to set up a custom PubMed search form that communicates with the PubMed website through XML and returns all entries directly in BibTeX format.

Technical Details

pubmedxml2bib is written in Kotlin as an illustrative demonstration how common Kotlin source code can be used to generate javascript as well as JVM binaries. It has been generated from scratch solely using a simple text editor, no IDE. A central Makefile allows the specification of the target platform (javascript or JVM) and generates the respective sources from common source code (most of the project) and a few platform specific functions.
